### 🚀 Featured Project: Veltrix V2
Saya sedang aktif mengembangkan **Veltrix V2**, sebuah mesin agen AI otonom lokal berkinerja tinggi:
- ⚡ **FastAPI ASGI Backend** dengan pemrosesan non-blocking & penanganan concurrency optimal.
- 💾 **SQLite WAL Mode** untuk crash-safety state recovery & ChromaDB untuk pencarian semantik lokal.
- 🔗 **Dynamic Fallback Routing**: Rantai alur kognitif dinamis dari Claude, Gemini, DeepSeek, hingga Ollama lokal.
- 🤖 **Stealth CLI & Computer Use**: Agen otonom yang mampu berinteraksi langsung di terminal dan OS menggunakan visual reasoning.
